Task Worker for Spring

The Task Worker is a Spring Integration module usable for situations where the actual work should not be done in the "main" application. For example, a web application should not generate and render a huge pdf document itself because of performance issues with Java Heap and Java PermGen size issues.

Note: Currently, the Spring Integration configuration supports only a RabbitMQ/AMPQ variant. Feel free to provide an alternative. This could be easily done with Spring profiles.

Getting Started
